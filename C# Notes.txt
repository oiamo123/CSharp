methods:
static allows you to access the method without having to create an object copy of it first ie:
class SomeClass {
	public int InstanceMethod() {  return 1;  }
	public static int StaticMethod() {  return 42;  }
}

someClass instance = new someClass()
instance.InstanceMethod() // fine
instance.StaticMethod() // Won't Compile

SomeClass.InstanceMethod() // Won't Compile
SomeClass.StaticMethod() // fine

refer to https://stackoverflow.com/questions/4124102/whats-a-static-method-in-c?rq=3

when passing arguments into methods, you can use "ref" as a suffix to reference a variable

double x, y;
Swap( ref x, ref y )

this method will change the original x & y variables.

private static void Swap(ref double a, ref double b)
{
	doube aux = a;
	a = b;
	b = aux;
}

Json get requests:

within async function:
using HttpClient client = new HttpClient();

try
{
    // Make the GET request
    HttpResponseMessage response = await client.GetAsync("http://127.0.0.1:8000/api/register");
    response.EnsureSuccessStatusCode();

    // Read the response content as a string
    string jsonResponse = await response.Content.ReadAsStringAsync();

    // Parse the JSON response
    var jsonObject = JsonNode.Parse(jsonResponse).AsArray();
    Console.WriteLine(jsonObject[0]);
}
catch (Exception ex)
{
    Console.WriteLine($"An error occurred: {ex.Message}");
}

 - Delegates are used in LINQ
 - Common use is to create programmer defined events
 - To use a delegate, you create it's instance and define a method that it will use.
